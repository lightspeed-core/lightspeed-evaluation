# Library Mode - Programmatic API Usage

Use LightSpeed Evaluation as a Python library for real-time integration.

## Run Example

```bash
# From project root
export OPENAI_API_KEY="your-key"
uv run python examples/01_getting_started/library_mode/example.py
```

## Key Features

- Programmatic configuration
- Create evaluation data in code
- Access results directly
- No YAML files needed
